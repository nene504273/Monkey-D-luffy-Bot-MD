## 🍖 **`DESCRIPCIÓN`**

> **Monkey D. Luffy Bot** es un bot multifuncional para WhatsApp inspirado en la libertad y poder del futuro Rey de los Piratas. Diseñado para administrar grupos, descargar multimedia y entretener a tus nakamas con estilo RPG y anime.

## 🏴‍☠️ INSTALACIÓN EN TERMUX

> Sigue los comandos **uno por uno**.

<details>
<summary><b>⚡ CLIC PARA VER COMANDOS</b></summary>
<br>

### 1. Dar permisos a Termux  
```bash
termux-setup-storage
```

### 2. Actualizar paquetes  
```bash
apt update
```

```bash
apt upgrade -y
```

### 3. Instalar dependencias  
```bash
pkg install git -y
```

```bash
pkg install nodejs -y
```

```bash
pkg install ffmpeg -y
```

```bash
pkg install imagemagick -y
```

```bash
pkg install yarn -y
```

### 4. Clonar repositorio  
```bash
git clone https://github.com/nene504273/Monkey-D-luffy-Bot-MD
```

```bash
cd Monkey-D-luffy-Bot-MD
```

### 5. Instalar librerías  
```bash
yarn install
```

```bash
npm install
```

### 6. Iniciar el bot  
```bash
npm start
```

</details>
